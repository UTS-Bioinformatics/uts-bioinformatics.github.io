## Packages

### [TMASplit](https://github.com/UTS-Bioinformatics/TMASplit)

**TMASplit** is an R package for separating tissue microarray cores from spatial transcriptomics datasets stored as Seurat objects.

It helps remove cells/debris outside tissue cores and assign accurate core boundaries, making TMA-based spatial transcriptomics analysis cleaner and easier.

[View TMASplit on GitHub](https://github.com/UTS-Bioinformatics/TMASplit)
